tldr:
  - Lade eine Lightning-Wallet herunter (Phoenix, Muun oder Wallet of Satoshi)
  - Empfange Bitcoin via Lightning-Invoice oder -Adresse
  - Sende Zahlungen durch Scannen von QR-Codes oder Einfügen von Invoices
  - Transaktionen werden in Sekunden abgewickelt mit nahezu null Gebühren
  - Fang klein an beim Lernen, dann skaliere hoch
whyPeopleAsk: Menschen hören, dass Lightning die Lösung für Bitcoins Skalierungsherausforderungen ist, wissen aber nicht wie man es tatsächlich nutzt. Dies ist der praktische Einstiegs-Guide.
steelmanObjection: "Lightning ist zu kompliziert für normale Menschen. Wenn Bitcoin eine zweite Schicht braucht die technisches Wissen erfordert, wird es nie Massenadoption erreichen."
whatIsTrue:
  - Lightning ist mit modernen Wallets viel einfacher geworden
  - Etwas technische Komplexität wird vor Nutzern verborgen
  - Sofortige Zahlungen mit nahezu null Gebühren funktionieren gut
  - Adoption wächst, besonders in Entwicklungsländern
whatIsUncertain:
  - Wie viel weiter die UX vereinfacht werden kann
  - Ob Lightning die dominante Zahlungsschicht sein wird
  - Wie Regulierung Lightning-Wallets beeinflussen könnte
